# Advanced REST Client import-export UI module

A module containing ARC data export UIs.

## Development

```sh
git clone https://github.com/@advanced-rest-client/arc-ie
cd arc-ie
npm install
```

### Running the demo locally

```sh
npm start
```

### Running the tests

```sh
npm test
```
